Recommandation Films

by Jeremie et Ludovic

Nous vous présentons notre site internet vous fournissant des recommandations de films.
Notre algroithme propose trois moteurs de recommandations qui vous sont exposés juste après.

Un algorithme de recommandation est une forme spécifique de filtrage de l'information
Cet algorithme vise à présenter des contenus qui sont susceptibles d'intéresser l'utilisateur


Votre recommandation:

Moteur Item Base (les films similaires),

Utilisation des informations sur les préférences passées des utilisateurs pour recommander des films qui pourraient vous intéresser.

Nous utilisons la corrélation de Pearson pour mesurer la similarité entre deux films, le principe fondateur est que si deux films ont une évaluation semblable dans la base de données (4.88 étoiles pour le film XXX et 4.91 étoiles pour le film YYY) il est probable que si vous aussi attribuer une évaluation proche pour un des deux films vous apprécierez le second.

Moteur User Base (Les utilisateurs similaires),

Comparaison de vos réponses avec celle des utilisateurs de la base de données ayant des réponses très proches et ainsi vous proposer les films que ces utilisateurs ont appréciés puisque vous avez des goûts similaires.

Pour ce moteur, nous utilisons la similarité cosinus pour trouver les utilisateurs les plus proches de vous.

Moteur KNN (les plus proches voisins),

La base de ce moteur de recommandation est un algorithme d’apprentissage automatique qui analyse les préférences des utilisateurs. On recherche les films similaires aux films que vous avez le mieux notés, en analysant la distance entre ces films.
